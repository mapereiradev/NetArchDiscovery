About

NetarchDiscovery helps security analysts and sysadmins discover and monitor their Linux network infrastructures. It leverages powerful tools like Nmap and Gobuster, presenting results in a unified web interface.

Features

  • Run customized Nmap scans and view structured results.
  • Launch Gobuster directory brute-forcing.
  • Analyze discovered services, users, permissions and detect common vulnerabilities.
  • View detailed reports with recommendations for hardening.
